Project Description:- 
	This project is about management of customers orders and it's a backend application for Admin to handel with the customers & their orders.
	It provides detailed reports of	customers, orders and order placed by particular customer. 

	This project have several features & functionalities :- 
	Features:- 
		- Add New Customer, 
		- Delete Customer, 
		- Update and Edit Customer, 
		- List of all customers & orders.
	Functionalities:-
		Customer Module 
			- Customer Header, Customer Footer,
			- Customer Card, List & Table View, 
			- Customer Info, Customer filter
			- Models of Customer and their Address,
			
		Order Module
			- Order Header, Order Footer,
			- List of all orders, Customer Order, Oder filter

		About, Login,
		Server 
			- Contains Customer & Order JSON,
			- Contains Rest API to fetch Customer & Order JSON using multiple parameters.
